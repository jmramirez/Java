All *.java files which are NOT members of a named package should appear
in this directory.

Once named packages have been introduced, all *.java file belonging to
any named package should appear in a sub-directory of this directory whose
name is the same as the name of the package concerned. For example, the
source file for a class which is specified to be in package business
should be exist ONLY in a directory named /sources/business

ALL source files you create should be valid ASCII TEXT files.  In
particular, they should be created by an industry standard text-editor
and and be processable other text utility programs.

NON ASCII TEXT FILES ARE NOT ACCEPTABLE.

Please see the note in /data/README.TXT for further details.
